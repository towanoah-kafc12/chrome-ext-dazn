# Implementation Policy Template

使う技術、設計方針、実装方針、スコープ、テスト方針を保存するためのテンプレートです。  
アイデア詳細化後、実装計画に入る前にこのファイルを埋め、AIエージェントが実装時に参照できる状態にします。
