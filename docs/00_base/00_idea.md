# Idea Detailing Template

アイデアだけの状態から、実装に進める粒度まで詳細化するためのテンプレートです。  
要件がまだ曖昧な依頼を受けた場合は、すぐに実装計画へ進まず、まずこのテンプレートを使ってアイデアを整理します。

## 使い方

1. ユーザの最初のアイデアを `元のアイデア` にそのまま残す。
2. AIエージェントが不足している観点を洗い出す。
3. 質問は一度に出しすぎず、原則 3〜5 個に絞る。
4. 回答がない項目は、必要に応じて `仮説` として置く。
5. 実装に関わる技術・設計・方針は `docs/00_base/01_implementation.md` に転記する。
6. 未確定の内容は `docs/01_pending` に転記する。
7. 合意できた内容は `docs/02_decision` に転記する。
8. レビューで見つかった改善点は `docs/03_review` に転記する。
