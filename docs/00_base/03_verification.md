## 検証対象

### 今回確認する範囲

- `app/manifest.json` がChrome拡張のManifest V3定義として読み込めること。
- `app/content.js` に構文エラーがないこと。
- `app/content.css` がDAZNページ上の `video` 要素と周辺要素へ横幅調整と高さ上限を適用すること。
- Chromeへ `app/` をローカル拡張として読み込む手順。

### 今回確認しない範囲

- Chrome Web Store公開。
- DAZNログイン後の全ページ種別での網羅確認。
- DAZN固有DOMに合わせた精密なセレクタ調整。

### 前提条件

- ChromeまたはChromium系ブラウザを使う。
- `chrome://extensions/` でデベロッパーモードを有効にする。
- `app/` を「パッケージ化されていない拡張機能を読み込む」から選択する。
- DAZNの動画再生ページは、ユーザがログイン済みの環境で確認する。
